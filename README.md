# The Seam - ICA

## Getting Started

__Install npm packages__
```sh
$ npm install
```

__Build the project__
```sh
$ npm run build
```

## Development

> Note: Be sure to rebuild the project before pushing your changes, to make them show on the dev site.

__Run either of the following__
```sh
$ npm run dev
```

```sh
$ npm run start
```
